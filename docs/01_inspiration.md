# Product Inspiration

* A copy editor that feels like a google doc, the hemmingway app, or a notion doc. Easy to freeform write in document mode. The document should contain sections of markdown.
* Each section of markdown is *versioned* and can be easily swapped out
* A wire frame tool - that is not meant to be a full design, just layout and components. Similar to Relume (TODO WHAT ELSE)

The tool uses AI and smart defaults to "magically" link the copy to the wireframe. The user can also manually link, but it should feel like magic.

## Copy Editor Inspiration

The copy editor interface should be simple and intuitive to get started. It should feel like a google doc or notion page.

Some apps that we love to use for reference for the copy editor:

* Google docs - simple, easy, and intuitive text editors for website copy
* [Hemingway app](https://hemingwayapp.com/) - perfect for the writing and editing copy side of things
  * What we love: clean writing page. Easy to see before/after edits when it makes a suggestion.
  * We also love the different "view" modes. Our app will have three modes:
    * 1) Copy mode, which is like writing a document in a google doc
    * 2) Wireframe mode: just the wireframe with the copy inside of it
    * 3) Side-by-side, which shows both models next to each other so you can see how they connect, and edit either side by side.
* Notion - nice sections, easy to get started

## Wireframe Inspiration

* [Relume Library Wireframes](https://www.relume.io/)
  * What we love about relume are the greyscale components and their wireframe editor. It's a tool I use often as a web designer, but not practical for making copy edits while you are still in the creative writing or brainstorming phase
  * Unlike relume, we don't need to deal with brand style guides. We want the wireframe to be a simple greyscale design system. Eventually we may want to swap design systems, so make this part of the code modular and swap-able.
